# Функциональное и логическое программирование

## Лекция 1

**Императивность** - мгновенное выполнение функции(выполняется сразу).

**Функционально программирование** - парадигма программирования, в которой процесс вычисления трактуется как вычисление значений функций в математическом понимании последних (в отличие от функций как подпрограмм в процедурном программировании).

**Виды функций:**

- Чистые

- Рекурсивные

- Специальные функции или формы(не детерминированы)

- Псевдо функции - спецэффекты

- Функции с вариантами значений

- Функционалы - работа с функциями в аргументах

- Базисные: 

  ```lisp
  car, cdr, cons, apply, funcall, cond, quote, eval, lambda, list
  ```

**Способы определения функций:**

1. ? - определение

   ```commonlisp
   (lambda ?-список форма)
   ```

   где 

   - ?-список - аргументы(формальные параметры)

   - форма - что делать

   ```commonlisp
   (defun f ?-выражение)
   ```
